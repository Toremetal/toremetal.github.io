MB-Tools: Windows Image Media Manager.

    MB-Tools is designed for Creating Virtual Machines offline, Network Deployment Images, and Backing Up and Restoring Hard Drive Images for Drive Swaps.

Automatic VHD & Virtual Machine Creator

    Select the size of the vhd to create and the installation media to use and let the app do the rest.

Important

    This is an Administrator Application. By default the application installs in standard user state to protect your computer from malware potentially misusing this application. In order to actually run the operations you need to right click and select run as Administrator on the (Admin) shortcut in the Start Menu. The other shortcut runs in standard mode only. Use the standard mode shortcut to check for and download updated versions of the application. The new version will create a new (admin) shortcut in the start menu. The old shortcut will still remain as the previos version will not be uninstalled incase you need to go back it an update has errors. If you want to remove the old shortcut from the start menu right click it and select open file location and delete the link.
